Overview
This project analyzes the London Bike Share dataset (August 2023) to uncover usage patterns, fleet performance, station demand, and operational insights.
The analysis includes data cleaning, exploratory analysis, visualization, and business‑oriented conclusions.

Technologies Used:
Python
Pandas
NumPy
Matplotlib / Seaborn
Jupyter Notebook

Dataset source: https://www.kaggle.com/datasets/dongxiaol/londonbikejourneyaug2023
